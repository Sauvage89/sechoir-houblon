# Ce site est instancier via une appli php

Il faut aller dans le dossier : `sechoir-houblon/serveur_web/test/html/`  
Est executer la commande : `php -S localhost:8000`

Dans notre cas en test le fichier index est en php mais en production il seras en html.
Dans la Raspberry Pi l'application Apache2 gére trés bien php
