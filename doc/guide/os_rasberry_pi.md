# Instalation de l'os rasberry pi sur une carte SD

### Logicielle nécessaire :
- Rasberry Pi Imager

### Matérielle necessaire :
- Carte SD > 16GB
- Support carte SD

### Note
	Pour l'instalation de l'os Rasberry pi il faut brancher la carte SD avec le support SD sur le pc
	Il faudras que la SD soit vierge
	Il faut lancer le logicielle Rabsrry Pi Imager et suivre les étape d'instalation
	La carte SD contient un os Rasberry pi

### Etape d'instaltion de l'imager
	1 : Appareil : choisir l'appareil correpondant ou vas être plug la carte SD (Rasberry Pi 3)
	2 : OS : Choisir une version de l'os (64-bit)
	3 : Stockage : Selectionner la carte SD comme élément de stockage
	4 : Personalisation - Nom d'hôte : Donner un nom d'hote (rasbpiSechoir)
	5 : Personalisation - Localisation : Donner une capitale, un fuseau horaire et le type de clavier (Paris(France), Europe/Paris, fr)
	6 : Personalisation - Utilisateur : Donner un nom d'utilisateur et son mot de passe (user_sechoir, password)
	7 : Personalisation - Wi-Fi : Selectionner Reseau securisé et donner un SSID et le mot de passe du point wifi (sechoir, password)
	8 : Personalisation - Accès a distance : Ne pas activer le SSH
	9 : Personalisation - Raspberry Pi Connect : Ne pas activer le Raspberry Pi Connect
	10 : Ecriture : Validation des élèment renseigner et formatage de la carte
	11 : Terminé : La carte SD contient un os Raspberry Pi
